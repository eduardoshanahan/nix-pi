# `docs/`

Repository-level documentation under `nix-pi/` lives here.

- `lifecycle/`
  - stable host setup, provisioning, secrets, and remote-build docs
- `policy/`
  - stable host policy, divergence, and confidentiality docs
- `recovery/`
  - backup and rebuild runbooks

Use `../README.md` for the repo overview and `../DOCUMENTATION_INDEX.md` for
the quick navigation index.
